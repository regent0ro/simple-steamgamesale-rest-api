ゲームリスト取得(GET)

アドレス

http://localhost:{PORT}/game

リクエスト

なし

レスポンス

HTTPレスポンスコード：成功時は200、エラー時は400

成功時のレスポンスボディ：DBに登録されているゲームの配列

要素 説明
id ゲームID
name ゲーム名
rel_date ゲームがリリースされた日
price 通常価格
discounted_price 現在の割引価格
discount_per 割引率

Githubリンク

https://github.com/regent0ro/simple-steamgamesale-rest-api/blob/main/src/server.js

ゲーム追加(POST)

アドレス

http://localhost:{PORT}/game

リクエスト

登録に必要な情報

要素 説明
name ゲーム名(必須)
rel_date ゲームがリリースされた日(必須)
price 通常価格(必須)
discounted_price 現在の割引価格(必須)
discount_per 割引率

レスポンス

HTTPレスポンスコード：成功時は201、エラー時は400

成功時のレスポンスボディ：IDを含めたDBに登録されたゲームの情報

要素 説明
id ゲームID
name ゲーム名
rel_date ゲームがリリースされた日
price 通常価格
discounted_price 現在の割引価格
discount_per 割引率

Githubリンク

https://github.com/regent0ro/simple-steamgamesale-rest-api/blob/main/src/server.js
