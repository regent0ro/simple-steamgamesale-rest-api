STEAM game sale DEMO

説明

STEAMのゲームセール情報を元に簡単なREST APIを試すことができるデモです。各API説明はこちら。

また、Chromeブラウザを基準に作成されました。

触ってみる

GET(現在のゲームリストを取得)

POST(新しいゲームを追加、追加後GETのリストが更新される)

PATCH(idを指定して変更データを入力、追加後GETのリストが更新される)

DELETE(ゲームの削除、追加後GETのリストが更新される)
